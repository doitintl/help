# Infirma light

This page tests the CSS color variables used by the Infirma light-mode theme.

:::seealso

- [Infirma docs: Utilities: Colors](https://infima.dev/docs/utilities/colors)
- [GitHub: Infirma variables definition](https://github.com/facebookincubator/infima/blob/main/packages/core/styles/common/variables.pcss)

:::

:::hint

Hover over any color in the swatch tables (including the cell background colors) to
see the corresponding CSS variable name.

The first column uses a checkerboard pattern as a standard neutral for the alpha chanel.
