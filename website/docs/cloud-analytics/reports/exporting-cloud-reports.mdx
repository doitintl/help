# Export reports

After saving your report, you can export it to various formats.

:::important

Required Permissions: **Cloud Analytics**

:::

## Export a report

To export a report, select the _Export report_ button (the download icon) from the right-hand menu on the report page.

<picture
  img={require('../../../assets/png/cmp-report-export-button.png')}
  alt="A screenshot highlighting the export report button"
  frame="drop-shadow"
/>

## Options

You have the following options when exporting a report:

- Export to CSV
- Copy CSV to clipboard
- Export to Google Sheets
- Download PDF
- Download PNG

:::important

If it's your first time exporting a report to Google Sheets, you may be asked to give CMP permission to create Google Drive files.

:::
