# Activate Flexsave

There are two steps to activate Flexsave for GCP, you first need to set up your billing.

## Provide payment information

Provide the payment information as shown on the screen, including:

- Billing address
- Company details: Company name and Tax ID
- Contact details: First name, last name, email address, and phone number
- Payments: payment method and billing currency

A billing profile is created when you save the changes.

## Activate Flexsave for GCP

Flexsave needs access to your cloud environment to add/remove savings plans/committed used discounts

1. Enable discount sharing.

   For detailed instructions, refer to Google Cloud Docs: [Turning on committed use discount sharing](https://cloud.google.com/compute/docs/instances/signing-up-committed-use-discounts#turning_on_committed_use_discount_sharing).

2. Provide the location of your billing data.

Flexsave attaches GCP projects with CUDs to your billing, thus providing optimal coverage for your workloads.

Get your GCP account information

1. Find your organization ID: [Getting your organization ID](https://cloud.google.com/resource-manager/docs/creating-managing-organization#retrieving_your_organization_id)
2. Find your project ID that contains your BigQuery billing data. [Identifying projects](https://cloud.google.com/resource-manager/docs/creating-managing-projects#identifying_projects)
3. Find your dataset ID that contains your BigQuery billing data. Refer to Google Docs on [Getting dataset information](https://cloud.google.com/bigquery/docs/dataset-metadata#getting_dataset_information), the dataset name is referred to as the datasetId.
4. Find your table ID that contains your BigQuery billing data. Refer to Google Docs about how to [Get table information](https://cloud.google.com/bigquery/docs/tables#get_table_information), the table name is referred to as the tableId.

Run the commands.

Test the connection.

Now you can click ACTIVATE FLEXSAVE to activate the feature.
